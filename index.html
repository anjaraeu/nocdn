This is a FuckCDN instance

FuckCDN is a alternative to Content Delivery Network on the Internet. To make it brief, a FuckCDN contains 10 CDNs content stored locally to be independent of CDNs.

Why ?

  • Privacy

    CDNs operating by enterprises like Google, MS and Cloudflare doesn't really explain the using of access logs.

  • Fast

    Less requests to external servers are better for page load time.

  • Look

    You look anarchic while using FuckCDN, and this is freakin' cool !
